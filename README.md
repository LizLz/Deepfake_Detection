Deepfake Detection 
Authors: Zheng, Felipe

Course: Team Lab - Phonetics Track

The project is meant to detect deepfake using asv2019 LA dataset. Our research questions are:
- Can a shallow CNN achieve acceptable performance for the task?
- Which acoustic features best detect spoofing?
- How do feature processing techniques impact model performance?

Quick Start

- You can download the required dataset from [https://www.asvspoof.org/index2019.html]
